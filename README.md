# Master en APIs RESTful con NodeJS – 3 Backends Completos

🚀 Este repositorio contiene los proyectos desarrollados durante el curso [Master en APIs RESTful con NodeJS](https://www.udemy.com/course/master-en-apis-restful-con-nodejs-crea-3-backends-completos/), impartido por Víctor Robles. Se construyen tres APIs RESTful completas aplicando conceptos avanzados de desarrollo backend con Node.js, Express.js y MongoDB.

---

## 📁 Proyectos incluidos

### 1. API REST para Blog

- **Descripción**: API que permite la gestión de publicaciones de blog, incluyendo creación, lectura, actualización y eliminación de entradas.
- **Tecnologías utilizadas**:
  - Node.js
  - Express.js
  - MongoDB
  - Mongoose

### 2. API REST para Red Social

- **Descripción**: API que permite la gestión de usuarios y publicaciones en una red social simulada.
- **Tecnologías utilizadas**:
  - Node.js
  - Express.js
  - MongoDB
  - Mongoose

### 3. API REST para Aplicación de Música

- **Descripción**: API que permite la gestión de artistas, álbumes y canciones, simulando una aplicación de música.
- **Tecnologías utilizadas**:
  - Node.js
  - Express.js
  - MongoDB
  - Mongoose
